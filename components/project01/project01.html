Booking Website

Full-stack webpage for hotel booking management with a focus on usability and full-stack architecture.

TECHNOLOGY

ABOUT THE PROJECT

This project involves the development of a web application for managing hotel bookings, integrating a modern front-end and a functional back-end architecture. Its main objective is to offer a seamless and engaging experience for both customers who wish to make bookings and the administrator in charge of managing the site's information.

Which was my personal motivation?

I decided to develop this project as a way to capture my knowledge and progress in web development. My main motivation was to create a comprehensive project that encompasses both the frontend and backend, allowing me to document my progress, learnings, mistakes, and improvements throughout the process. I also hope this project will reflect my capacity for continuous learning and demonstrate my skills to future recruiters or companies interested in my professional profile.

The project is currently in the development phase. New pages, features, and visual improvements are being progressively added, with the goal of optimizing the user experience and expanding the system's technical capabilities.

Which are the specific objectives or goals of the project?

• Put my knowledge of HTML, CSS, and JavaScript into practice, developing a website from scratch.
• Thoroughly understand the structure and logic of a complete web project before incorporating frameworks or libraries.
• Develop a clean, functional, and responsive interface that improves the user experience.
• Document the development process to demonstrate my work methodology, problem-solving skills, and technical progress.

Passionate full-stack web developer with hands-on experience across diverse projects, specializing in both frontend and backend development.
